# Sunray for Alfred

## How to compile 'Sunray for Alred'
Run this on your Alfred:

```
cd ~
git clone https://github.com/Ardumower/Sunray.git
cd ~/Sunray/alfred/build
rm -Rf *
cmake ..
make
```
